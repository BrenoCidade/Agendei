# Agendei – MVP de Agendamento Online

Aplicação web para agendamento online de serviços. O usuário pode escolher serviço, data e horário de forma simples e rápida.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como rodar o projeto localmente

Pré-requisitos:

- Node.js (LTS) e npm instalados.

Passos:

```sh
npm install
npm run dev
```

O Vite irá subir o servidor de desenvolvimento (por padrão em `http://localhost:5173` ou porta semelhante indicada no terminal).
